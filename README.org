* .emacs.d
My Emacs Configurations.

* 核心配置文件结构
- init.el :: 主入口文件
- site-lisp/ :: 模块化配置目录
- site-lisp/config/ :: 模块化配置文件存放目录
- site-lisp/extension/ :: 扩展文件存放目录

- site-lisp/config/init-config.el :: 模块化配置入口配置文件
- site-lisp/config/template.el :: 模块化配置模板文件
- site-lisp/config/init/ :: 各个模块初始化配置文件存放目录

- site-lisp/extension/themes/ :: 主题扩展目录
